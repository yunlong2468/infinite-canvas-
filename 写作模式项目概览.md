## 前端核心功能

### Activity Bar（左侧栏）
- 宽度 64px，图标 56px×40px 居中
- 📖 大纲 / 💬 对话 / 🧠 Skill配置 / ⚙️ API配置
- Ctrl+1~4 快捷键切换

### 多窗格分屏
- 每个窗格 = 独立标签栏(44px) + 内容区
- 窗格间可拖拽分割线调整宽度（最小 280px）
- 标签可拖拽重排和跨窗格移动
- 虚影横向跟随鼠标，其他标签平移让出空隙
- 右键标签：全屏模式 / 向右拆分 / 关闭
- Zen Mode：全屏 + 浮动退出按钮 + ESC 退出

### 大纲面板
- 卷/章树形结构，双击章节打开编辑器标签
- "生成大纲"按钮调用生成Agent

### 对话面板
- 多Agent群聊系统：🎭调配师/📋大纲/👥角色/💬对话/🕷️爬虫/🔍审核/🧠技能优化
- 发送消息自动调用调配师 LLM
- @Agent 下拉菜单选择
- 选项按钮：调配师回复中 `- [按钮文字]` 独立行自动渲染为可点击按钮
- 按钮状态持久化（pickedOption字段防刷新重置）
- 思考冒泡：三个青色圆点跳动动画替代静态文字
- 消息时间戳：每条消息显示 HH:MM
- 未读追踪：lastReadMsgIndex 索引追踪，非 DOM 计数
- 输入栏：Textarea 自动增高(max 120px)，Enter发送/Shift+Enter换行

### 用户消息右键菜单
- 📋 复制：全文复制到剪贴板
- ✏️ 编辑：气泡内原地编辑，文本框+红色取消/青色确认按钮，确认弹窗
- ↩ 撤回：仅最新一条用户消息可用，智能体思考中不可用
- 撤回后显示"你撤回了一条消息，重新编辑"
- 重新编辑填充到输入框（不自动发送）
- 重新发送后提示保留但链接消失

### Skill 配置面板
- CRUD + 启用/禁用
- 完整编辑表单：中文名称/英文名称/描述/SKILL.md内容(200px)/参考JSON(120px)
- 卡片显示 content 前100字预览

### API 配置面板
- 每个Agent独立配置：供应商/API地址/API Key/模型
- 选择DeepSeek自动填入 https://api.deepseek.com
- 根据已保存endpoint自动识别供应商

### 编辑器面板
- 工具栏：标题/粗体/斜体/下划线/对齐/列表
- 自动保存（1秒防抖）

## 调配师系统提示词要点

- 只负责采访/整理需求/协调，不亲自生成内容
- 选项必须用 `- [按钮文字]` 独立行格式，一个选项一行
- 按钮数量必须覆盖所有给出的选项，不能只列部分
- 按钮文字 ≤20字
- 提问放在按钮上方作为结尾句
- 全文 ≤300 字
- 一个场景一句话概括核心冲突，不要铺陈细节
- 禁止排比堆砌："不再压制A/不再压制B/不再压制C"这类禁止
- 不要替用户做决定，始终征求确认
